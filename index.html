Introduction

Video description is a common application of video understanding models and an intuitive task to evaluate the capabilities of video understanding models. However, existing video captioning datasets typically pair each video with a single-sentence caption, which is too short to stress-test the full capacity of models. To this end, DREAM-1K is proposed as a challenging Video Description Benchmark features in Rich events, Actions, and Motions. It contains: 1) 1,000 video clips with diverse complexities from five different origins: live-action movies, animation movies, stock videos, YouTube videos, and TikTok-style short videos. 2) A fine-grained description for each video clip labeled by skilled annotators. As traditional automatic evaluation metrics such as CIDEr for video captioning are not competent for evaluating the quality of detailed video descriptions, DREAM-1K takes AutoDQ (Automatic Description Quality) as a more interpretable automatic evaluation approach, which has been proven to be discriminative in practice and shows a high degree of consistency with manual evaluation.

Evaluation Approach

We propose AutoDQ as a more interpretable approach to automatic video description evaluation to overcome the shortage of traditional automatic evaluation metrics (such as CIDEr) in evaluating detailed video descriptions. AutoDQ uses an extraction model to extract events from two video descriptions, then uses an entailment model to examine how many events extracted from one description are entailed by the other description. The ratio of events in reference description (human annotation) that are entailed by model-generated description is defined as Recall, and vice versa is is defined as Precision. We use ChatGPT (gpt-35-turbo-0125) to implement both models (See evaluation scripts), as shown bellow.
